Dining Dash

Dining Dash is a mobile on-campus food ordering experience. Using Axure, a prototype was conceived and evolved after careful examination of task analyses in user experience studes, user questionnaires, and original wireframes. Upon final evaluation, feedback from test users indicated the interface was quick and easy to use. Created for the User Interface Design class at Georgia Tech (CS 3750). View Final Presentation »
